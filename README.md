📝 Lista de Tarefas

Projeto feito em **HTML, CSS e JavaScript** para gerenciar tarefas do dia a dia.

✨ Funcionalidades

📝 Adicionar novas tarefas com descrição

✅ Marcar tarefas como concluídas ou pendentes

🔄 Trocar tarefas de lugar entre si

✏️ Editar tarefas existentes

🗑️ Excluir tarefas da lista

🚀 Como usar

Abra o arquivo `index.html` no navegador

Digite a tarefa no campo de entrada e clique em "Adicionar"

Use os botões da interface para marcar como concluída, editar, trocar de lugar ou excluir tarefas

🛠️ Tecnologias

- HTML
- CSS
- JavaScript

💡 Observações

Projeto para estudo e prática de desenvolvimento web, focado em manipulação de DOM e lógica de gerenciamento de listas.
